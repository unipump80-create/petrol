# Petrol — мониторинг цен и наличия топлива на АЗС г. Иваново

**Что:** сервер + веб для розничного пользователя (водителя): найти где дешевле и где есть нужное топливо.

**Аудитория:** розница (водители).
**Данные:** парсинг агрегаторов (+ опц. платный API benzup).

---

## Ограничения (важно)

- **Остатки в литрах недоступны** публично. «Наличие» в MVP = виды топлива (92/95/98/100/ДТ/газ) + статус работы АЗС.
- **Легал:** уважать robots.txt, разумная частота (раз в 3-6 ч), кэш, без DDoS. Проверить ToS агрегаторов — возможен запрет на парс. benzup отдаёт лицензионный API как легальная альтернатива.

## Источники (проверены 2026-06-24, Phase 0)

| Источник | Роль | Статус проверки |
|---|---|---|
| **russiabase** (ОСНОВНОЙ) | АЗС + per-station цены + координаты + адреса | ✅ данные в Next.js `__NEXT_DATA__` на SSR-странице → **JS/Playwright НЕ нужен**. city=154051 (Иваново), пагинация. 27 АЗС, реальные цены, наличие (null=нет) |
| OSM Overpass | опц. обогащение: часы работы | ✅ 39 АЗС, bbox, opening_hours. В MVP не подключён (нужен fuzzy-join по координатам) |
| ~~fuelprice.ru~~ | отклонён | ❌ смешанное качество: точные данные Газпромнефти, но круглые заглушки (90/95) у Татнефти/Лукойла |
| benzup API | отклонён | ❌ платный |

**Ключевой вывод (Phase 2):** russiabase отдаёт всё в одном чистом JSON — цены **по каждой станции** (точнее, чем по бренду: 61.88 vs 63.36 внутри Газпромнефти), координаты, адреса, наличие топлива. Связка цен с координатами по `poiid`.

## Стек

Python 3.12 · FastAPI · httpx (regex по `__NEXT_DATA__`) · APScheduler · SQLite + SQLAlchemy · Яндекс.Карты JS.
selectolax/Playwright не понадобились.

## Модель данных

- `stations`: id, poiid(russiabase), osm_id(опц.), brand, name, address, lat, lon, opening_hours, fuel_types(JSON)
- `fuel_types`: справочник кодов (ai92, ai95, ai98, ai100, diesel, gas + premium-варианты)
- `prices`: station_id(FK), fuel_type, price, observed_at, source — **цена по станции**

Наличие вида = `stations.fuel_types` (топливо с ненулевой ценой в russiabase).
