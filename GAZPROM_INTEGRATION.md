## 🚀 Как собрать APK

### Локально (для тестирования)
```bash
# 1. Запустить сервер
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

# 2. Если телефон в другой сети, использовать ngrok
ngrok http 8000
# Получить HTTPS URL: https://xxxx-xx-xxx-xx.ngrok.io

# 3. На PWABuilder (https://www.pwabuilder.com)
# - Загрузить URL
# - Скачать Android APK

# 4. На телефон
adb install -r petrol.apk
```

### Production (Render/Heroku)
```bash
# Если деплоено на Render:
git push origin main
# (Render автоматически собирает из git)

# Затем в PWABuilder загрузить URL деплоя
# https://petrol-xxxxx.onrender.com
```

## 📱 Функционал в приложении

1. **Все АЗС (режим по умолчанию)**
   - 51 станция г. Иванова
   - Сводка по городу: мин/средн/макс цены
   - Поиск по типу топлива
   - Сортировка по цене или названию

2. **Только Газпромнефть (кнопка 🔵)**
   - 15 АЗС Газпромнефти
   - Отдельная статистика
   - Все цены обновляются в реальном времени
   - Работает сортировка и фильтры

## 🔧 Интеграция с источниками

- **russiabase.ru** — основной источник (27+ АЗС, включая Газпромнефть)
- Обновление каждые 6 часов (APScheduler)
- Все данные в локальной SQLite БД
- Работает offline-first (PWA Service Worker)

## ✨ Технические детали

- **БД:** SQLite, 3 таблицы (Station, Price, FuelType)
- **API:** FastAPI с CORS
- **Frontend:** Vanilla JS + Leaflet.js для карты
- **PWA:** Service Worker для offline + обновлений
- **Парсинг:** httpx + regex для __NEXT_DATA__ из russiabase
- **Планировщик:** APScheduler для фоновых обновлений
