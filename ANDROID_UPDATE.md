# Обновление Petrol на Android после изменений

## 🔄 Если приложение уже установлено

### Способ 1: Очистить кэш приложения (самый быстрый)
```
Телефон → Настройки → Приложения → Petrol
→ Память → Очистить кэш
→ Вернуться в приложение → Обновить 🔄
```

Или в самом приложении:
- Нажать кнопку 🔄 Обновить в правом верхнем углу
- Дождаться обновления данных

### Способ 2: Переустановить приложение (полное обновление)
1. Удалить приложение
2. Собрать новый APK (см. ниже)
3. Установить заново

## 📦 Собрать новый APK

### Локально (если есть сервер)
```bash
# 1. Запустить сервер
cd C:\UNIPUMP\AI\AI-project\petrol
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000

# 2. Проверить что работает
curl http://localhost:8000/
# Должна вернуться HTML страница с кнопкой Газпромнефть

# 3. Если тестируем с мобилки (другая сеть):
#    Установить ngrok: https://ngrok.com/
ngrok http 8000
# Скопировать HTTPS URL (вида https://xxxx-xxxx-xxxx.ngrok.io)

# 4. На компьютере с мобилкой:
#    Перейти на https://www.pwabuilder.com
#    - Вставить URL в поле (localhost:8000 или ngrok URL)
#    - Нажать Start
#    - Выбрать Android
#    - Нажать Build
#    - Скачать petrol.apk
```

### На Render/Heroku (если деплоено)
```
1. git push origin main  (обновит автоматически)
2. Дождаться деплоя (5-10 минут)
3. На PWABuilder загрузить URL деплоя
4. Build → Download APK
5. Установить на телефон: adb install -r petrol.apk
```

## 🚨 Если обновления всё ещё не видно

### Проверить версию приложения:
Открыть DevTools в браузере (F12):
```
Console → вводим:
fetch('/version').then(r => r.json()).then(d => console.log(d.version))
```

Должна вернуться версия с hash коммита (0.2.0+xxxxx)

### Если версия старая:
1. Полностью закрыть приложение (не в фоне)
2. Очистить кэш (см. Способ 1 выше)
3. Открыть заново
4. Дождаться загрузки (может быть медленно)

### Если совсем не помогает:
1. Удалить приложение
2. Очистить данные:
   Телефон → Настройки → Приложения → (поиск) "Petrol" → Удалить
3. Перезагрузить телефон
4. Установить новый APK

## 🔧 Технические детали

- PWA кэширует файлы в Service Worker
- Версия из git-хеша в `app/main.py` (функция `_resolve_version()`)
- Manifest обновляется при каждом деплое
- На Render версия берётся из `RENDER_GIT_COMMIT` переменной

## 💡 Советы

1. **Для быстрого обновления данных:** нажать 🔄 в приложении (не очищать весь кэш)
2. **Для обновления приложения:** очистить кэш приложения в Настройках
3. **Для полного переустановки:** удалить и переустановить APK

Всё остальное обновляется автоматически! 🎉
