# Полный список features Petrol

## 📊 Данные

### Источники данных
- ✅ **RussiaBase** — основной (51 АЗС, обновление каждые 6 часов)
- ✅ **Card-Oil.ru** — наличие топлива из статического JSON (без цен, без Playwright)
- ✅ **Переключение источников** — `DATA_SOURCE=russiabase|cardoil`

### Типы топлива
- АИ-92 (98% доступности)
- АИ-95 (100% доступности)
- АИ-98 (20%)
- АИ-100 (28%)
- Дизель (95%)
- Газ (28%)
- АИ-95+ (26%)

## 🔌 API

### Основные endpoints
- ✅ `GET /prices/summary` — сводка по городу
- ✅ `GET /stations` — список АЗС с фильтрацией
- ✅ `GET /stations/{id}` — детали станции
- ✅ `GET /prices/gazprom/availability` — статистика Газпромнефти
- ✅ `GET /prices/gazprom/locations` — локации Газпромнефти

### Система и мониторинг
- ✅ `GET /version` — версия приложения
- ✅ `GET /health` — здоровье данных
- ✅ `GET /prices/source` — информация об источнике
- ✅ `GET /prices/stats` — полная статистика
- ✅ `GET /prices/cache/stats` — статистика кэша

### История и графики
- ✅ `GET /history/{station_id}/{fuel_type}` — история цен за период
- ✅ Сохранение истории в БД
- ✅ Статистика (min, max, avg) по истории
